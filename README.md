# Netflix_SQL_Analiz
Netflix içeriklerinin güncel analizi ve veri temizlenmesi.

Bu proje, Netflix'in ham veri setinin SQL Server kullanılarak analiz edilmesi ve temizlenmesi sürecini içerir.

## 🛠 Kullanılan Teknolojiler
- **SQL Server (T-SQL)**
- **SQL Management Studio**

## 🚀 Proje İçeriği
- **Veri Temizliği:** `ISNULL` ve `VIEW` yapısı kullanılarak NULL değerlerin yönetimi.
- **Trend Analizi:** İçeriklerin yıllara göre üretim hızı.
- **Kategori Analizi:** `STRING_SPLIT` ve `CROSS APPLY` ile tür bazlı ayrıştırma.
- **Performans Metrikleri:** Yönetmen verimliliği ve süre analizi.
- **Yaş Grupları:** İçeriklerin hangi yaş gruplarına hitaben üretildikleri.
- **Ülke Bazlı Analiz:** Ülkelerin içerik üretim hacmi.
